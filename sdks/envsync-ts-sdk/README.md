# envsync-ts-sdk

This is generated SDK for EnvSync API developed by EnvSync Cloud

## Install

```sh
npm install @envsync-cloud/envsync-ts-sdk
```

## Release

- published to npm as `@envsync-cloud/envsync-ts-sdk`
- released from the monorepo tag workflow on tags like `v0.4.1`
- package version matches the repo tag version without the `v` prefix
